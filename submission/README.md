# Submission to the competition

Copy the model checkpoint to the directory, as well as the config and the 
directory with the model code. By default, the `baseline` directory with a 
minimal PPO setup is used in `main.py`.

Be sure to manually change the checkpoint name and config name in the 
script to your own. Finally, tar all the files with
```
tar -czvf submission.tar.gz *
```
and submit it to the competition.
